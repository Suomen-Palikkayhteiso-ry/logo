# Suomen Palikkaharrastajat ry — Brand Guide

Brand assets for Suomen Palikkaharrastajat ry. The logo mark is a LEGO minifig
head rendered as brick art using the `brick_blockify.py` pipeline.

## Brand Colors

| Color | Hex | Rebrickable ID |
|-------|-----|----------------|
| **LEGO Yellow** | `#F2CD37` | #14 |
| **LEGO Black** | `#05131D` | #0 |
| **LEGO White** | `#FFFFFF` | #15 |

Edit `colors.py` to change the palette; all outputs are regenerated by `make`.

## Logo Variants

### Square (profile picture / icon)

| Variant | Background | File stem |
|---------|------------|-----------|
| Yellow *(recommended)* | Light | `minifig-yellow` |
| Black | Light | `minifig-black` |
| White | Dark | `minifig-white` |

Files: `logo/square/svg/`, `logo/square/png/` (800 px PNG + WebP)

### Horizontal (with association name subtitle)

Same three colour variants, with "Suomen Palikkaharrastajat ry" underneath.

Files: `logo/horizontal/svg/` (`*-full.svg`), `logo/horizontal/png/`

## Build

Requires Python 3 with **Pillow** and **cairosvg** (provided by `devenv.nix`).

```bash
make          # build everything (design SVGs → brick SVGs → PNG/WebP)
make designs  # regenerate design/*.svg from colors.py
make logos    # brick SVGs only
make pngs     # PNG + WebP exports only
make clean    # remove all generated files
```

## Pipeline

```
colors.py
    │
    ▼
generate_designs.py  →  design/minifig-*.svg   (200×200 source art)
                              │
                              ▼
                       brick_blockify.py  →  logo/square/svg/minifig-*.svg
                                         →  logo/horizontal/svg/minifig-*.svg
                                              │
                                              ▼
                                       compose_logo.py  →  logo/horizontal/svg/minifig-*-full.svg
                                              │
                                              ▼
                                       cairosvg + Pillow  →  PNG + WebP exports
```

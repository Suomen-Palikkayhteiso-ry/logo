Lataa materiaalit

Kaikki tiedostot generoidaan automaattisesti make-komennolla.

Tuotetut tiedostot

  • 3 neliölogoa – keltainen, musta, valkoinen (SVG + PNG + WebP, 800 px)
  • 3 logoa alaotsikolla – sama kolmikko alaotsikon kanssa (SVG + PNG + WebP)
  • Outfit-fontti – muuttuva fontti (SIL Open Font License)

Värien muuttaminen

Muokkaa tiedostoa colors.py ja aja make. Kaikki logot päivittyvät automaattisesti uusilla väreillä. Rebrickable-värikoodit: rebrickable.com/colors

Build

make          # build all SVG + PNG/WebP
make logos    # SVG only
make designs  # regenerate design SVGs from colors.py
make clean    # remove all generated files
